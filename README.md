## 스터디 진행

1. 각 과제 개인학습
2. 과제 별 `issue`에 질문 2~3개 작성
3. 담당 발표 과제 준비
4. 담당 과제 브랜치 생성, (예시 : `subject/issue-#01`)
5. 해당 과제 폴더(`subject_01`)에 발표자료 추가하여 브랜치 push  
   (블로그 글로 작성한 경우 `README.md`에 링크 추가)
6. 온라인 발표 및 질의응답 내용 `README.md`에 정리 및 브랜치 push
7. pull request 및 merge
